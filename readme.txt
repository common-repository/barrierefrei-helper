Verbessert die Barrierefreiheit auf Websites mit Anpassungen wie Schriftgrößenänderung, Farbwahl, und Kontrastmodus.

== Description ==
Das Plugin "Barrierefrei Helper" verbessert die Zugänglichkeit von WordPress-Seiten durch Bereitstellung folgender Funktionen:
– Schriftgrößenänderung
– Farbwahl
– Kontrastmodus
– Dynamisches Inhaltsverzeichnis

Diese Funktionen sind speziell darauf ausgelegt, mit dem Elementor Page Builder zu interagieren, wobei die globalen Farbvariablen von Elementor für die Farbanpassung genutzt werden.

== Installation ==
1. Stellen Sie sicher, dass Elementor auf Ihrer WordPress-Seite installiert und aktiviert ist, sollten Sie die Farbwahl benutzen.
2. Laden Sie die ZIP-Datei des Plugins "Barrierefrei Helper" herunter.
3. Gehen Sie in Ihrem WordPress-Dashboard zu "Plugins" > "Installieren".
4. Klicken Sie auf "Plugin hochladen", wählen Sie die heruntergeladene Datei und klicken Sie auf "Jetzt installieren".
5. Nach der Installation klicken Sie auf "Plugin aktivieren".

== Configuration ==
1. Navigieren Sie nach der Aktivierung zum Admin-Menü "Barrierefreiheit", um die Einstellungen des Plugins zu konfigurieren.
2. Aktivieren Sie die gewünschten Funktionen wie Schriftgrößenänderung, Farbwahl, Kontrastmodus und Inhaltsverzeichnis.

== Frequently Asked Questions ==
= Wie aktiviere ich die Farbwahl? =
Die Farbwahl kann in den Plugin-Einstellungen aktiviert werden, sofern Elementor als Page Builder verwendet wird.

= Was mache ich, wenn die Einstellungen nicht gespeichert werden? =
Stellen Sie sicher, dass Sie nach jeder Änderung der Einstellungen auf "Speichern" klicken.
